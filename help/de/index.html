Kurzbeschreibung

ExpoSong ist ein freies, Open Source Präsentationsprogramm mit dem Ziel Worship so einfach wie möglich zu machen. Erstellen Sie Liedtexte oder reine Text-Präsentationen, erstellen Sie Ablaufpläne und werfen Sie Folien an die Leinwand.

Ablaufpläne

Ablaufpläne werden verwendet um die Reihenfolge von Präsentationen festzulegen.

Um einen Ablaufplan zu erstellen wählen Sie im Menü Ablaufplan->Neu. Geben Sie den gewünschten Namen Ihres Ablaufplans ein. Fügen Sie Lieder zum Ablaufplan hinzu, indem Sie diese von der Bibliothek in den Ablaufplan ziehen.

Eine gute Idee wäre, wenn nötig, die alten Ablaufpläne zu löschen.

Präsentationen erstellen

Präsentationen sind eine Sammlung von Folien, die auf dem Bildschirm präsentiert werden sollen. Um eine neue Präsentation zu erstellen wählen Sie im Menü Präsentation->Neu. Wählen Sie dann im Untermenü den gewünschten Präsentationstyp aus.

Jeder Abschnitt im Editor steht für eine Folie.

Die Text-Präsentation wird verwendet um nicht-spezifische Textfolien zu erstellen. Sie kann für Predigten, Ankündigungen oder andere Informationen eingesetzt werden.

Die Lieder-Präsentation ist für Liedtexte gedacht. Sie verwendet Folien-Marker, wie z.B. "Verse #", "Chorus", "Bridge" oder "Refrain", die das Präsentieren wesentlich vereinfachen. Es gibt einen Informationen-Tabulator, deßen Daten zusammen mit der CCLI Nummer am unteren Bilschirm-Rand angezeigt werden.

Die Bilder-Präsentation wird verwendet um Bilder zu präsentieren. Fügen Sie mehrere Bilder auf einmal hinzu, indem sie im Auswahl-Dialog die Shift- oder Str-Taste drücken. ändern Sie die Reihenfolge der Bilder durch drag&drop.

Mitteilungen

Möglicherweise müßen sie jemandem im Saal über etwas informieren, z.B. ein weinendes Baby im Kinderraum. Dies kann mit einer Mitteilung gemacht werden. Die Mitteilung kann in der Textbox unter dem Vorschau-Fenster eingegeben werden. Drücken Sie die Enter-Taste oder den zweiten Button neben der Textbox um die Mitteilung sichtbar zu machen. Der Button neben der Textbox wird die Mitteilung wieder löschen.

Hintergründe

Das Hintergrundbild von Präsentationen kann entweder ein Farbübergang oder eine Bild-Datei sein. Ändern Sie das Bild indem Sie die die Buttons links neben dem Vorschau-Bildschirm benutzen.

Tips zu Bildern

Hintergrundbilder sollten generell einen weichen Fokus und einen schwachen Kontrast haben. Hintergründe mit harten Konturen oder mit vielen Farben machen das Lesen des Textes schwer.
